# mnist_digit_classifier
A CNN-based handwritten digit classifier built on the MNIST dataset using TensorFlow/Keras with a descriptive walk-through

## Description
This project builds a CNN to recognize handwritten digits (0-9).
- Framework : TensorFlow/Keras
- Dataset : MNIST
- Architecture : Conv2D -> MaxPooling2D -> Flatten -> Dense -> Softmax
- Accuracy : 97.96%
